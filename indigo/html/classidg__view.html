◆ add_filter()

idg_view::add_filter ( string  $function,
callable  $fun = null 
)

Adds a filter with a given name.

It is not an error if the filter already exists. If $fun is not specified, $function is treated as a function name, otherwise $fun will be called. The functions must have the following signature: 

 function filter(idg_view $view, string $text): ?string;

If the filter returns null it is ignored.

If a namespace is set in the view, the filter will be looked for in that namespace.

Parameters
string$functionThe name of the filter function
callable$funA closure to execute as the filter
