◆ get_property()

idg_object::get_property ( string  $name,
  $execute_hooks = true 
)

Returns the value of a property, possibly generated by a hook function if it is not explicitly set, or null.

If $execute_hooks is true and a hook function is set, the return value of the hook function is returned. If this fails, the same is tried with the associated idg_type's hook function.

Parameters
string$nameThe name of the property
bool$execute_hooksWhether to execute hooks at all
Returns
string|null The value of the property
