◆ __construct()

idg_object::__construct ( )

Constructs an idg object and assigns the corresponding idg type.

The idg_object_type is created if necessary, otherwise taken from a list.

$idg_id is assigned a unique id based on the (php) object type and the number of objects constructed so far.

Reimplemented in idg_view_html_slot, idg_view_html_item, idg_view_html_renderer, idg_view_html_container, idg_view_html_part, idg_view_html_element, idg_view_html, idg_view, idg_tree_node, idg_document, idg_folder, idg_site_element, idg_site, idg_renderer_declaration, and idg_datasource_declaration.
